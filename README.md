Saavn Music Player (React Native)

A modern music player built using React Native and Expo.

Features

Search songs using Saavn API

Inline play/pause from Home screen

Full player screen with:

Large artwork

Play / Pause

10s Forward / Backward

Next / Previous

Queue management:

Add to queue

Remove from queue

Reorder songs

Mini player with playback controls

Background audio playback support

Tech Stack

React Native (Expo)

Zustand (State Management)

Expo AV (Audio playback)

React Navigation

Ionicons
